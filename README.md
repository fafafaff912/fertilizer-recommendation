#  Генерация рекомендаций по удобрениям на основе состояния поля

## Описание задачи

Multi-output regression: по состоянию почвы, погоде и истории урожая
модель рекомендует оптимальные дозы удобрений (N, P, K).

**Вход:** pH почвы, влажность, температура, осадки, тип культуры,
предыдущий урожай, текущее содержание N/P/K в почве.

**Выход:** рекомендуемые дозы Nitrogen, Phosphorus, Potassium (кг/га).

## Цель и мотивация

- Оптимизация расхода удобрений → снижение затрат на 15-30%
- Предотвращение избыточного внесения → экологическая безопасность
- Повышение урожайности за счёт точных рекомендаций
- Доступный AI-инструмент для фермеров

## Данные

- **FAOSTAT** — [Fertilizer Use by Crop](https://www.fao.org/faostat/en/#data/RFN)
- **Дополнительно:** [Fertilizer Prediction Dataset (Kaggle)](https://www.kaggle.com/datasets/gdabhishek/fertilizer-prediction)
- При недоступности API — генерируются реалистичные синтетические данные
  на основе агрономических формул и статистик FAO

### Признаки (входы):

| Признак | Описание | Единица |
|---------|----------|---------|
| temperature | Средняя температура | °C |
| humidity | Влажность воздуха | % |
| rainfall | Осадки | мм/мес |
| soil_ph | pH почвы | 0-14 |
| soil_moisture | Влажность почвы | % |
| soil_N | Содержание азота в почве | кг/га |
| soil_P | Содержание фосфора | кг/га |
| soil_K | Содержание калия | кг/га |
| organic_carbon | Органический углерод | % |
| prev_yield | Урожай прошлого сезона | т/га |
| crop_type | Тип культуры (encoded) | категория |
| area | Площадь поля | га |

### Целевые переменные (выходы):

| Выход | Описание | Единица |
|-------|----------|---------|
| N_recommended | Доза азота | кг/га |
| P_recommended | Доза фосфора | кг/га |
| K_recommended | Доза калия | кг/га |
